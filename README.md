## Installation

The app requires [Node.js](https://nodejs.org/) 14.6.0+ to run.

Get project from repo and run:

```sh
https://github.com/Ilya-Korsak/car_safety_rating_web_app.git
cd ./car_safety_rating_web_app
npm install
npm run dev
```

To build...

```sh
npm run build
npm run start
```
